# public

## Purpose
Next.js が静的配信する SVG アイコンファイル群。SNS の UI で使用するアイコン（ハート・ブックマーク・コメント・通知・ホーム等）とブランドロゴを格納する。

## Key Files

| File | Description |
|------|-------------|
| `heart.svg` / `heart-fill.svg` | いいね（未・済） |
| `bookmark.svg` / `bookmark-fill.svg` | ブックマーク（未・済） |
| `comment.svg` | コメント |
| `notification.svg` | 通知 |
| `home.svg` | ホーム |
| `explore.svg` | 探索 |
| `paper-airplane.svg` | DM・送信 |
| `next.svg` / `vercel.svg` | フレームワークロゴ |

## For AI Agents

### Working In This Directory
- SVG を追加する場合は `src/components/` から `<img src="/icon-name.svg">` で参照
- Next.js の `public/` はルートパス `/` に直接マッピングされる
